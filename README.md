# 2D-DFT-Surface-Texture-Analysis
Early research on surface texture characterization using 2D-DFT. Once presented at Ecole Centrale de Lyon and the University of Cambridge.

## Documentation
The full slide deck is available here:
- English: [2D_DFT.pdf](docs/2D_DFT.pdf)
- Japanese: [2ddft_user_guideJP20180312.ppt](docs/2ddft_user_guideJP20180312.ppt)

## Overview
This repository contains my early research (1998) on surface texture evaluation 
using 2-dimensional discrete Fourier transform (2D-DFT). 
The study includes:

- [Artificial surface models](ca://s?q=Artificial_surface_models)
- [Analysis of real measurement data](ca://s?q=Real_measurement_data)
- [Signal and noise separation](ca://s?q=Signal_noise_separation)
- [Feature extraction of dimples](ca://s?q=Feature_extraction_dimples)
- [Cross-hatch analysis](ca://s?q=Cross_hatch_analysis)
- [Shot condition vs dimple characteristics](ca://s?q=Shot_condition_dimple_characteristics)
